# Code of Conduct

## Our Pledge

We pledge to make participation in **aieos-engineering-execution** a respectful, inclusive, and harassment-free experience for everyone.

This project is a public, community-driven repository focused on improving AI-assisted software delivery practices. We expect all participants to engage constructively and professionally.

---

## Our Standards

Examples of behavior that contribute to a positive environment include:

- Being respectful and considerate in communication
- Focusing on ideas, not individuals
- Providing constructive, actionable feedback
- Accepting responsibility for mistakes
- Showing empathy toward other contributors

Examples of unacceptable behavior include:

- Harassment, discrimination, or abusive language
- Personal attacks or insults
- Publishing others’ private information
- Trolling, spamming, or deliberately disruptive behavior
- Sustained unconstructive criticism

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues and pull requests
- Discussions and comments
- Documentation contributions
- Any other interaction related to this project

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

They may take appropriate action in response to behavior that violates these standards, including but not limited to:

- Issuing a warning
- Removing content
- Temporarily or permanently restricting participation

Decisions are made to protect the health and integrity of the project.

---

## Reporting

If you experience or witness unacceptable behavior, please report it to the project maintainers.

Reports should include:
- A description of the incident
- Relevant links or screenshots (if applicable)
- Any other context that may be helpful

Reports will be handled discreetly and respectfully.

## Final Note

This project values clear thinking, disciplined process, and respectful collaboration.

If you’re unsure whether something crosses a line, err on the side of professionalism and kindness.
