# Validator Output Examples

This directory contains **sample validator outputs** for the end-to-end example.

These files demonstrate:
- What a PASS looks like
- What a FAIL looks like
- How validators report blocking issues
- How completeness scores are expressed

All outputs:
- Follow the standardized validator schema
- Contain no suggestions or redesign
- Reference only missing or ambiguous content

---

## Included Validators

- PRD Validator
- ACF Validator
- SAD Validator
- DCF Validator
- TDD Validator
- WDD Validator
- Definition of Ready (DoR) Validator

---

## How to Use These Examples

- Compare your validator output to these samples
- Use them to train AI agents on expected behavior
- Use them to explain validator enforcement to stakeholders

These outputs are illustrative, not exhaustive.
